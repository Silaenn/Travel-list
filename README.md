# 🌴 Far Away — Pack Smart, Travel Light

A clean, interactive React packing list app to help you prepare for your next trip. Add items, track your progress, and never forget a travel essential again.

## Features

- **Add Items** — Add items with custom quantities to your packing list.
- **Manage Packing Status** — Toggle items as packed or unpacked with a simple checkbox.
- **Sort Items** — Sort your list by input order, description, or packed status.
- **Delete Items** — Remove individual items or clear the entire list.
- **Progress Tracking** — Real-time progress bar and percentage showing how packed you are.
- **Persistent Storage** — Your list is automatically saved to local storage and survives page refreshes.
- **Responsive Design** — Works across desktop, tablet, and mobile screens.

## Tech Stack

- React 19
- Vanilla CSS

## Getting Started

1. Clone the repository
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
